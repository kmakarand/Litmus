Analysis of Test:

The analysis of the test given can be got by clicking on
Exam -> Performance ->Performance Analysis

The following screen appears on the browser:



Now the candidate can see the number of tests he had attempted on the screen.From here the candidate can analyze the questions & its answers.Now if the candidate has to see the analysis of the exam he/she had given, he/she has to click on "Analyse" button.In the drop down menu there are five options 1. All answers,2. Right answers,3. Unattempted answers,4. Wrong Answers.The candidate has to select one option and then has to click on "Analyse" button.

The following screen appears:



Graphical View then he could get the graphical analysis of the exam given .
